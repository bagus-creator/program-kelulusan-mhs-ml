# 🎓 Program Kelulusan mahasiswa (Logistic Regression)

## 📌 Deskripsi Proyek
Proyek ini bertujuan untuk memprediksi apakah seorang mahasiswa akan **Lulus** atau **Gagal** berdasarkan nilai tes simulasi mereka. Menggunakan algoritma **Machine Learning (Logistic Regression)**, model ini memberikan probabilitas kelulusan secara akurat.

## 🛠️ Teknologi yang Digunakan
- **Python 3.x**
- **Pandas:** Manipulasi data tabel.
- **Scikit-Learn:** Pembuatan model Machine Learning.
- **NumPy:** Pengolahan array numerik.

## 📊 Dataset
Data terdiri dari fitur:
- `Nilai Simulasi`: Input angka dari 0-100.
- `Status Lulus`: Label kategori (1: Lulus, 0: Gagal).

## 🚀 Hasil Prediksi
Model berhasil memberikan prediksi interaktif. Contoh hasil:
- Input Nilai: **72**
- Prediksi: **Lulus**
- Peluang Kelulusan: **90.88%**
